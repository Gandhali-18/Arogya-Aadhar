# 🏥 Aarogya Aadhar

Aarogya Aadhar is a mobile-first digital literacy and healthcare awareness platform that helps users understand and 
safely use Indian government healthcare schemes like ABHA and Ayushman Bharat (PM-JAY).
It focuses on education, scam prevention, and accessibility for elderly and low digital literacy users.

---

## 🚀 Features

- **ABHA Sandbox**: Step-by-step simulation of ABHA number creation with no real data used  
- **ABDM / PM-JAY Explorer**: Eligibility check, scheme understanding, and hospital lookup  
- **Red Flag Detector**: Interactive scam detection using real-world scenarios  
- **Community Siren**: Report scams, upload screenshots, and spread awareness through community posts
